Gem Minner

Get the right value with the least digs!

Instruction

  • You will be given a random target value at the start of the game
  • There are four types of gems below. By clicking on a gem you will add a specific amount of points to the total value you get.
  • You win the game by matching the total value you got to the random target value.
  • You lose the game if the total value you got goes above the random target value.
  • The value of each of the gem is hidden. Guess it by clicking and checking the value monitor.
  • Every time when the game starts, the game will change the value of each gem!
